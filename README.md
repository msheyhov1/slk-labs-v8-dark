# SLK-labs v8-dark — Living System (скин v7)

Сайт-доказательство студии: **сам является живой системой**. Настоящая телеметрия,
настоящий realtime, настоящие health-check'и — ничего нарисованного.

Дизайн: скин v7 «живое поле» — тёмная подложка, костяная база, сигнальный зелёный как свет; энсо собирается сетью зелёным штрихом. Сестринская версия с дизайном «Тушь·Рассвет» — ../slk-labs-v8.
Оттиск энсо уникален для каждого посетителя (seed: время+viewport+GPU, хеш локально).

## Запуск

```bash
npm i
npm run dev          # http://localhost:3000
```

Все живые фичи прогрессивные: без ENV сайт полностью работает. Подключение
реалтайма/консьержа/статусов — `.env.example`.

## Что внутри

- **Живое поле** — WebGL2 без three: сеть узлов дрейфует, на загрузке собирает
  знак энсо кистью (давление, рваный край), потом живёт фоном. `enso --reseed`
  в терминале пересобирает штрих.
- **Boot-sequence** — реальные строки: GPU, вьюпорт, TTFB, вес документа.
- **HUD телеметрии** — FPS (rAF), GPU, ядра, сеть, LCP; «Ваш {GPU} · держим 60 fps».
- **Cmd+K** — палитра команд + терминал (`help`, `ls works`, `status`, `whoami`,
  `hire`, `enso --reseed`).
- **Окна ОС** — кейсы открываются живым сайтом в sandbox-iframe (драг, максимайз);
  фоллбеки для Mini App и приватных систем.
- **Живой курс** — USDT/RUB с Binance в карточке «Дозора» (edge-прокси, кэш 60с).
- **Контакт-сцены** — самолётик по Безье → Telegram, конверт со штемпелем → буфер,
  AI-консьерж (Claude, ключ только серверный).
- **Presence** — PartyKit: счётчик «на сайте сейчас» + анонимные курсоры (≤15, 30 Гц).
